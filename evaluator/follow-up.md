# Follow-Up Evaluation

The user has seen your initial evaluation and has a specific question before deciding. The previous evaluation and the user's question are included below.

## Your task

1. **Answer the question directly.** Don't restate the full evaluation — the user already saw it.
2. **Update your recommendation if the answer changes the picture.** If it doesn't change anything, say so.
3. **Be brief.** 2-4 sentences. The user wants to decide and move on.

## Output format

**[Answer to their question — 2-4 sentences]**

**Updated decision: [PURSUE / MAYBE / PASS — or "No change: still [DECISION]"]**
