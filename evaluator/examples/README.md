# Evaluation Examples

This folder contains real examples of job evaluations using the Pursuit evaluators.

## What's Here

Each example shows:
- **LinkedIn Intelligence Report** - Network analysis, poster credibility, timing signals, entry strategy
- **Job Fit Analysis** (if applicable) - Direct/Adjacent/Stretch categorization
- The decision made (Pursue/Maybe/Pass)
- Reasoning and next steps
- Outcome (if pursued - interview? offer? rejection?)

## Why Examples Matter

Seeing real evaluations helps you:
- Understand how network intelligence drives decisions
- See what "warm bridge" vs "cold outreach" looks like in practice
- Learn how to identify timing signals and urgency indicators
- Calibrate when to pursue based on access (not just fit)
- See the evaluator's reasoning process

## Anonymization

When contributing examples:
- **Company names:** Keep them (public job postings)
- **Connection names:** Anonymize (use "Connection A", "Connection B", etc.)
- **Poster names:** Anonymize or use generic titles
- **Personal background:** Keep high-level, remove identifying details
- **Message content:** Summarize context, don't quote verbatim

## Contributing Examples

If you use the evaluators and want to share your results:

1. Create a new markdown file: `YYYY-MM-DD-company-role.md`
2. Include:
   - LinkedIn Intelligence Report (anonymized connection names)
   - Job Fit Analysis (if you ran Step 2)
   - Your decision and reasoning
   - Next steps taken
   - Outcome (if you pursued)
3. Submit a PR or open an issue with your example

See existing examples for format reference.
